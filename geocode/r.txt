Summary
I am looking for a data engineer to extract, process, and geocode approximately 4,413 records from the Pandektis (National Documentation Centre of Greece) database regarding settlement renamings.

Source URL: https://pandektis.ekt.gr/pandektis/handle/10442/4968/browse?type=title&sort_by=1&order=ASC&rpp=4413

Project Requirements:

Data Extraction: Scrape all individual entries from the list. Each entry page contains crucial metadata.

Required Fields (to be included in GeoJSON properties):

old_name_gr: Original name in Greek script.
old_name_en: Original name translated/transliterated to English.
new_name_gr: New name in Greek script.
new_name_en: New name translated/transliterated to English.
prefecture: The "Nomos" (Prefecture) where the settlement is located.
province: The "Eparchia" (Province) where the settlement is located.
renaming_date: The official date of the renaming act.

Geocoding: * Provide coordinates in Decimal Degrees (WGS84) with at least 5 decimal places.

Since "Old Names" are historical, please use the "New Name + Prefecture" for geocoding to ensure accuracy.

Final Deliverables:

Primary: A valid GeoJSON file (RFC 7946). Coordinates should be in [longitude, latitude] order.
Secondary: An Excel (.xlsx) file with the same data for manual auditing.
Encoding: All files MUST be in UTF-8 to preserve Greek characters.

Technical Note:
The website uses a handle system. Some entries might link to modern settlement records—please ensure the script follows these links to get the most accurate "New Name" and location.

How to apply:
Please provide a short sample (JSON or CSV) of 5 rows from the database to demonstrate your ability to handle Greek characters and geocoding accuracy.
